# GuessWho

A simple web‑based game that lets you guess a character based on a series of yes/no questions. The project includes a Node.js backend and a React frontend built with Vite.

## Features
- Random question generation
- Score tracking
- Responsive UI

## Setup
```bash
# Install backend dependencies
cd backend && npm install
# Install frontend dependencies
cd ../frontend && npm install
# Run development servers
npm run dev   # from each folder
```
